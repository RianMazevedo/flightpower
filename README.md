# Software de Parametrização do Controlador de Potência

![Interface do Software](software_gui.png)

Este software foi desenvolvido para realizar a **parametrização, monitoramento e análise**
do controlador adaptativo de potência para motores brushless.
A aplicação permite configurar parâmetros internos da placa, visualizar telemetria em
tempo real, registrar ensaios e realizar depuração durante o desenvolvimento. 


## Bibliotecas necessárias

- Python 3.10+
- DearPyGUI
- PySerial
- Threading
- Matplotlib
- TOML para arquivos de configuração
